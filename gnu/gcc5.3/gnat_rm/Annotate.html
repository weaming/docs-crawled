3.2 Annotate

There are three forms of this aspect (where ID is an identifier, and ARG is a general expression).

`Annotate => ID'
Equivalent to pragma Annotate (ID, Entity => Name);
`Annotate => (ID)'
Equivalent to pragma Annotate (ID, Entity => Name);
`Annotate => (ID ,ID {, ARG})'
Equivalent to pragma Annotate (ID, ID {, ARG}, Entity => Name);
