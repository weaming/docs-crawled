SSL Certs

We provide automated conversions. The output CA bundle file in PEM format (250KB) is available from here:

    HTTP from curl.haxx.se: cacert.pem

    HTTPS from github: ca-bundle.crt

The PEM file contains the datestamp of the conversion and we try to only convert if there's a change in either the script or the source file.

RSA-1024 removed

Around early September 2014, Mozilla removed the trust bits from the certs in their CA bundle that were still using RSA 1024 bit keys. This may lead to TLS libraries having a hard time to verify some sites if the library in question doesn't properly support "path discovery" as per RFC 4158. (That includes OpenSSL and GnuTLS.)

The last CA bundle we converted from before that cleanup: an older ca-bundle from github.

The conversion script mk-ca-bundle

The mk-ca-bundle tool converts Mozilla's cert bundle to PEM format, suitable for (lib)curl and others. Writtten by Guenter Knauf.

Convert from your local Firefox installation

You can also extract the ca certs off your Firefox installation, if you just have the 'certutil' tool installed and run the firefox-db2pem.sh script!
